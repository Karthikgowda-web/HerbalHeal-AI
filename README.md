# HerbaScan: AI-Powered Medicinal Plant Identification

HerbaScan is a high-performance MERN stack application integrated with a Deep Learning model to identify medicinal plants from images and provide localized Ayurvedic remedies.

## 🌿 Overview
This project combines modern web architecture with Artificial Intelligence to bridge the gap between traditional herbal knowledge and digital accessibility.

## 🚀 Architecture
- **Frontend**: React.js with Vite, Tailwind CSS, and Framer Motion for a premium Nature-Tech aesthetic.
- **Backend**: Node.js & Express API with MongoDB for data persistence.
- **AI Logic**: TensorFlow-based Convolutional Neural Network (CNN) for high-accuracy plant classification (30+ species).
- **Multilingual Support**: Supports English, Hindi, and Kannada for medicinal data and remedies.

## 📂 Project Structure
- `frontend/`: React application and UI components.
- `backend/`: Express server, API routes, and AI inference logic.
- `README.md`: Project documentation and deployment guide.

## 🛠️ Setup & Execution
1.  **Backend**:
    - `cd backend`
    - `npm install`
    - `node index.js`
2.  **Frontend**:
    - `cd frontend`
    - `npm install`
    - `npm run dev`
